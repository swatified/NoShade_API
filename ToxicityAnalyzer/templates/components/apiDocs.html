NoShad API Overview

Monitor and analyze product comments across your website with our powerful API endpoints.

Base URL: https://api.noshad.com/v1

Authentication

All API requests require an API key to be included in the header.

Authorization: Bearer YOUR_API_KEY

GET /comments

Retrieve comments for a specific product.

Parameters
  • product_id (required) - Unique identifier of the product
  • page (optional) - Page number for pagination
  • limit (optional) - Number of comments per page
